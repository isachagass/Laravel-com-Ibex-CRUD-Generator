## Projeto de Teste – Laravel com Ibex CRUD Generator

Este projeto foi desenvolvido como um ambiente de testes e aprendizado, com o objetivo de compreender o funcionamento do Ibex CRUD Generator no framework Laravel.

A aplicação utiliza o pacote para gerar automaticamente um CRUD completo a partir de uma tabela do banco de dados, permitindo analisar a estrutura dos endpoints, controllers, models e, principalmente, das telas geradas pelo pacote.

O foco do projeto foi entender como o Ibex organiza o fluxo de dados, a criação das interfaces administrativas e a integração entre back-end e front-end dentro do ecossistema Laravel, sem preocupação com personalizações avançadas.

Este repositório representa um exercício prático para consolidar conhecimentos em Laravel, uso de pacotes externos e geração automatizada de CRUDs.
